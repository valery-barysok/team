# team

Touch Instinct разрабатывает мобильные приложения для МегаФона, Сбербанка, Riot Games, Mail.ru, Meduza и других неравнодушных брендов. Входим в тройку лучших мобильных разработчиков России по версии рейтинга Clutch, Adindex, Ruward, Tagline, Рейтинга Рейтингов. Мы — первая команда заказной разработки из России, которую приняли в состав международной ассоциации Application Developers Alliance. Гугл думает, что мы молодцы, поэтому в 2017 году получили статус Google Developers Certified Agency.
